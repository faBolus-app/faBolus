Screenshot placeholders
=======================

Every *.svg in this folder is a labeled placeholder. To use a real screenshot,
replace the file's contents with your image. Two easy options:

1. Keep the .svg filename and drop in a PNG/JPG instead:
   - Save your screenshot as, e.g., hud.png in this folder.
   - In the referencing page, change  hud.svg  ->  hud.png .

2. Or overwrite the .svg with an exported SVG.

Frames (rounded phone/watch bezel + shadow) are applied by CSS via the
figure classes .cx2-shot.phone / .watch / .wide — you do NOT need to add a
device frame to the image itself; a plain screenshot is fine.

Suggested capture sizes: iPhone 6.1" portrait, Apple Watch 45mm, Garmin venu3s.
